## 使用说明

### 查看新的页面标题

1. 打开浏览器，访问系统首页：[http://newbb.test.bangbangvip.com/autodev/a46094c5-4cff-434b-a815-3e73fc6d39a6/](http://newbb.test.bangbangvip.com/autodev/a46094c5-4cff-434b-a815-3e73fc6d39a6/)
2. 页面加载完成后，在页面**左上角**即可看到新的标题"**使劲吹牛逼的系统**"
3. 浏览器标签页的标题也会显示"**使劲吹牛逼的系统 - 仪表盘**"
4. 进入创建任务、任务详情、编辑任务等子页面，左上角标题同样已是新的"**使劲吹牛逼的系统**"

### 如果没有看到新标题

如果页面仍然显示旧标题，请尝试以下操作：

1. **强制刷新页面**：按 `Ctrl + F5`（Windows）或 `Cmd + Shift + R`（Mac）强制刷新浏览器缓存
2. **清除浏览器缓存**：在浏览器设置中清除缓存和 Cookie，然后重新访问
3. **使用无痕模式**：打开浏览器的无痕/隐私模式窗口访问，确认是否为新版本
