## 三、详细操作指南

### 3.1 访问系统

### 3.2 任务仪表盘

仪表盘是您的主要操作界面，包含以下区域：

#### 顶部导航栏
- 左侧显示「任务管理系统」Logo
- 右侧有「+ 创建任务」按钮，点击进入创建页面

#### 统计卡片区
页面顶部显示 5 个统计卡片，分别展示：

| 卡片 | 含义 |
|------|------|
| 全部任务 | 系统中任务总数 |
| 待办 | 状态为"待办"的任务数 |
| 进行中 | 状态为"进行中"的任务数 |
| 已完成 | 状态为"已完成"的任务数 |
| 已逾期 | 截止日期已过但未完成的任务数 |

#### 筛选栏
位于统计卡片下方，提供以下筛选工具：

- **搜索框**：输入关键词，按任务标题或描述模糊匹配
- **状态筛选**：下拉选择"待办/进行中/审核中/已完成"
- **优先级筛选**：下拉选择"严重/高/中/低"
- **排序方式**：选择按"创建时间/截止日期/优先级/标题"排序
- **排序方向**：选择"升序"或"降序"

> **提示**：修改任意筛选条件后，列表会立即刷新。

#### 任务列表表格

| 列名 | 说明 |
|------|------|
| 任务名称 | 点击可跳转到任务详情页 |
| 优先级 | 以彩色标签显示（严重/高/中/低） |
| 状态 | 以彩色标签显示（待办/进行中/审核中/已完成） |
| 分类 | 显示分类标签（工作/个人/学习/健康/财务） |
| 截止日期 | 显示日期，若无截止日期则显示 "-" |
| 进度 | 进度条 + 百分比数值 |
| 操作 | 查看、编辑、删除按钮 |

#### 分页控件
当任务数量超过 15 条时，表格底部会显示分页按钮。

### 3.3 创建任务

点击右上角「+ 创建任务」按钮，进入创建页面：

1. **任务标题**（必填）：输入任务名称，不能为空
2. **任务描述**（选填）：输入任务的详细说明，支持多行文本
3. **优先级**：从下拉列表选择（严重/高/中/低），默认为"中"
4. **状态**：从下拉列表选择（待办/进行中/审核中/已完成），默认为"待办"
5. **分类**：从下拉列表选择（工作/个人/学习/健康/财务），默认为"个人"
6. **截止日期**（选填）：点击日期选择器选择日期和时间
7. **预计耗时**（选填）：输入预计完成所需的小时数，支持小数（如 2.5）
8. **标签**（选填）：输入标签关键词，多个标签用英文逗号分隔
9. **进度**：拖动滑块设置完成进度（0-100%）

填写完成后点击「创建任务」按钮提交。成功后会跳转回仪表盘页面并显示成功提示。

---

### 3.4 查看任务详情

在仪表盘的任务列表中，点击任务名称即可进入详情页。详情页分为两个区域：

#### 左侧主内容区
- 任务标题
- 优先级、状态、分类标签
- 进度条（显示完成百分比）
- 任务描述（如有）
- 编辑和删除操作按钮

#### 右侧信息栏
- 截止日期（含剩余天数提示）
- 分类
- 预计耗时
- 创建时间
- 更新时间
- 标签列表

> **剩余天数提示规则**：
> - 已逾期：红色显示"已逾期 X 天"
> - 0 天：显示"今天截止"
> - ≤ 3 天：橙色显示"剩余 X 天"
> - > 3 天：绿色显示"剩余 X 天"

---

### 3.5 编辑任务

有两种方式进入编辑页面：

1. 在仪表盘列表中点击「编辑」按钮
2. 在任务详情页点击「编辑任务」按钮

编辑页面会自动填充当前任务的所有信息，您可以修改任意字段后点击「保存变更」。修改成功后跳转回任务详情页。

---

### 3.6 删除任务

在仪表盘列表或任务详情页点击「删除」按钮，系统会弹出确认对话框：

> "确定要删除任务"[任务标题]"吗？此操作不可恢复。"

点击"确定"后任务将被永久删除，点击"取消"则放弃操作。

## 五、注意事项

1. **数据安全**：删除任务后数据不可恢复，请谨慎操作。
2. **必填字段**：创建和编辑任务时，「任务标题」为必填项，不可为空。
3. **时间格式**：截止日期使用本地时区，请确保浏览器时间设置正确。
4. **标签输入**：标签用英文逗号分隔，输入后会自动解析为标签数组。
5. **进度更新**：建议在任务推进过程中及时更新进度百分比。
6. **逾期提醒**：系统会自动识别已逾期的任务（截止日期已过且状态不是"已完成"）。
7. **刷新页面**：任务列表不会自动刷新，如需获取最新数据请重新进入页面或调整筛选条件。
8. **浏览器兼容**：建议使用 Chrome、Edge、Firefox 等现代浏览器访问。

## 六、常见问题

**Q：创建任务时提示"创建失败"怎么办？**

A：请检查网络连接是否正常，以及后端服务是否在运行。如问题持续，请联系系统管理员。

**Q：如何快速找到某个任务？**

A：使用仪表盘顶部的搜索框，输入任务标题或描述中的关键词即可快速定位。

**Q：任务数量很多时如何管理？**

A：可以结合状态筛选和优先级筛选缩小范围，也可以按创建时间或截止日期排序查看最紧急的任务。

**Q：标签有什么用途？**

A：标签可以帮助您对任务进行更灵活的分类和标记，例如"紧急"、"前端"、"设计"等，方便后续查找。
